#MOE
MOE

##Required Packages
flask
```
pip install flask
```
pymongo
```
pip install pymongo
```

##Installation
Run setup.py to set admin password
```
python setup.py
```
The desired password can be change by changing setup.py
```
regPass("<insert password here>")

```
Run app.py to start server
```
python app.py
```
